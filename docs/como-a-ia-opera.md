# Como a IA opera no Business OS

O sistema funciona melhor quando a IA segue poucas regras com disciplina.

## Regras centrais

- comece pelo dashboard ou home note mais proximo do problema
- busque primeiro na pasta mais estreita possivel
- abra apenas as notas necessarias para agir
- crie uma nota nova quando uma decisao, reuniao ou thread merecer vida propria
- evite arquivos acumuladores como `Meetings.md`, `Tasks.md` ou `Decisions.md`

## O que isso melhora

- menos leitura desnecessaria
- menos resposta generica da IA
- mais continuidade entre reunioes, follow-ups e execucao
- menos dependencia de memoria informal

## Como pensar o vault

- pastas definem ownership
- links definem relationships
- dashboards orientam navegacao
- notas atomicas guardam contexto duravel
