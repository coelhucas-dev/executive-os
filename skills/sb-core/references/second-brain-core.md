Leia `../../../spec/vault-spec.md` para o modelo público canônico.

Use este resumo dentro das skills:

- `01 Dashboards` são os pontos de entrada preferidos para retrieval.
- `10 People`, `20 Companies`, `30 Projects`, `40 Meetings`, `50 Content`, `60 Knowledge`, `70 Tasks`, `80 Routines` e `90 Personal Ops` são pastas de ownership.
- `95 Life Modules` é opcional.
- `99 Templates` contém templates públicos.
- Os agentes devem buscar de forma restrita e manter notas atômicas.
